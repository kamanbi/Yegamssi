**패키지명:** com.yegamssi.yegamssi

## 1. 앱 개요

예감씨는 날씨·야외활동점수·운세를 결합해 **오늘 외출을 해야 할지 판단하는 행동 판단 앱**이다.  
정보 제공이 아닌 **판단 지원**이 핵심 가치다.

| 구분 | 내용 |
|---|---|
| 최소 SDK | Android API 24 (Android 7.0) |
| 타겟 SDK | Android API 36 (Android 16) |
| 언어 | 한국어, 영어, 일본어(스텁), 중국어(스텁) |
| 광고 | AdMob 배너 + 전면 |

## 2. 핵심 기능 명세

### 2-1. 날씨

| 항목 | 내용 |
|---|---|
| 데이터 소스 | 기상청 KMA API (1순위) → Fallback 구조 |
| 조회 방식 | GPS 위치 기반 (권한 없을 시 마지막 캐시 → 서울 기본값) |
| 포함 정보 | 현재 기온·체감·습도·풍속·UV·강수확률·미세먼지(PM10·PM2.5)·오존 |
| 예보 | 시간별 24시간 + 주간 예보 (오전/오후 구분) |
| 야간 아이콘 | NOAA/Spencer 공식 기반 일출/일몰 자동 판별, 야간 4종 별도 아이콘 적용 |
| 캐시 | WeatherCacheStore (SharedPreferences), 앱 재실행 시 즉시 표시 후 백그라운드 갱신 |
| 갱신 주기 (앱 내) | 30분 타이머 자동 재조회 |

**야간 아이콘 적용 조건:**

| 조건 | 아이콘 |
|---|---|
| 맑음 + 야간 | night_sunny |
| 구름조금 + 야간 | night-cloudy |
| 안개 + 야간 | night-hazey |
| 폭염 + 야간 | night-hot (열대야) |
| 그 외 | 주간 아이콘 그대로 |

### 2-4. Android 홈 위젯

| 항목 | 내용 |
|---|---|
| 표시 정보 | 날씨 아이콘, 기온, 체감, 야외점수, 운세심볼, 날짜·시간 |
| 갱신 방법 | AlarmManager + Flutter WorkManager 이중 구조 |
| 갱신 주기 | 30분 (AlarmManager 정확 발화) |
| Doze 대응 | `setExactAndAllowWhileIdle()` 적용 |
| 부팅 후 복구 | BootReceiver로 알람 자동 재등록 |
| 운세 갱신 | 앱 포그라운드에서만 갱신 (백그라운드는 기존 심볼 유지) |

**위젯 갱신 흐름:**

```
AlarmManager 발화 (30분)
  → YegamssiWidget.refreshAll()      캐시 데이터로 즉시 UI 갱신
  → WorkManager.enqueue()            날씨 API 호출 (네트워크 조건)
  → AlarmScheduler.schedule()        다음 30분 알람 재등록
```

## 3. 주요 수정 이력 (이번 출시 기준)

### ① 야간 날씨 아이콘
- 일출/일몰 자동 판별 (외부 패키지 없이 NOAA 공식 직접 구현)
- 야간 아이콘 4종 적용 (앱 + 위젯 PNG 별도)

### ② 위젯 자동 갱신 안정화
- `AlarmManager.setExactAndAllowWhileIdle()` 도입
- Doze 모드·삼성 절전 모드에서 30분 정확 발화 보장
- BootReceiver로 재부팅 후 자동 복구
- WorkManager `onResume` 마다 타이머 리셋 버그 수정

### ③ 강수확률 감점 기준 재설계
- 기존: 70%↑ -40, 40%↑ -20, 20%↑ -10
- 변경: 60% 미만 0점, 10% 단위 누진 차감, 90%↑ 최대 -70

### ④ 즐겨찾기 지역 날씨
- 날씨 헤더 드롭다운 (상단 표시, 광고에 가리지 않음)
- 10초 자동 복귀, 즐겨찾기 5개 저장

### ⑤ 구글 플레이 인앱 업데이트
- `in_app_update 4.2.2` 연동
- 앱 시작 시 자동 확인 → 구글 플레이 UI 연결
