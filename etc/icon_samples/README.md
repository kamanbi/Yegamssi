# Weather Card Samples

`Wd_icon.png`를 기준으로 앱용으로 약간 단순화한 카드형 날씨 샘플이다.

## 포함 파일
- `weather_sunny_card_premium.svg`
- `weather_rain_card_premium.svg`
- `weather_cloudy_card_premium.svg`

## 유지한 요소
- 짙은 네이비 카드 배경
- 부드러운 구름 입체감
- 골드 하이라이트
- 차분한 광택과 얇은 외곽선

## 단순화한 요소
- 과한 파티클과 노이즈 제거
- 카드 내부 문구 제거
- 심볼 인지에 필요한 레이어만 유지

## 다음 확장 우선순위
1. partly_cloudy
2. heavy_rain
3. thunderstorm
4. snow
5. sleet
6. windy
7. hazy
