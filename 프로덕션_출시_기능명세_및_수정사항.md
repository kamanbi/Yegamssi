# 예감씨 프로덕션 출시 기능명세 및 중요 수정사항

## 1. 앱 개요
- 앱명: 예감씨
- 현재 버전: 1.1.8+40
- 목적: 날씨, 야외 활동 점수, 오늘의 운세를 함께 제공해 사용자가 하루의 흐름을 빠르게 판단하도록 돕는 앱
- 핵심 방향: 단순 날씨 조회가 아니라 오늘의 컨디션과 행동 판단을 지원하는 생활형 예측 앱

## 2. 주요 기능명세

### 2.1 날씨
- 현재 위치 기반 현재 날씨 표시
- 기상청 API 기반 현재 기온, 체감온도, 습도, 풍속, 강수확률, 강수량 표시
- 시간별 예보 24시간 표시
- 주간 예보 7일 표시
- 주간 예보의 오전/오후 날씨 아이콘과 온도 분리 표시
- 주간 예보의 강수확률 및 예상강수량 표시
- 날씨 캐시 지원
  - 앱 진입 시 기존 캐시를 먼저 표시
  - 신규 데이터 오류 시 기존 정상 데이터를 유지
  - 예보 데이터가 비어 있으면 기존 예보 캐시를 보존
- 위치 즐겨찾기 지원
- 현재 위치와 즐겨찾기 위치의 날씨 조회 지원

### 2.2 야외 활동 점수
- 날씨 데이터를 기반으로 야외 활동 적합도 점수 산출
- 반영 요소
  - 강수확률
  - 강수/눈/비 등 날씨 상태
  - 풍속
  - 기온
  - 미세먼지
  - 초미세먼지
  - 오존
  - 자외선
- 대기질 박스에서 일반 사용자가 이해하기 쉬운 항목명 사용
- 대기질 항목별 그래프 표시
- 위젯에는 앱에서 계산한 야외 점수 캐시를 사용
  - 위젯 전용 별도 산식 제거
  - 앱 점수와 위젯 점수 불일치 방지

### 2.3 운세
- 사용자 생년월일 및 생시 기반 오늘의 운세 제공
- 종합운세 및 카테고리별 운세 제공
- 운세 캐시 정책 유지
  - 하루 기준 재사용
  - 불필요한 반복 로딩 방지
- 운세는 날씨/점수 판단을 보조하는 서브 콘텐츠로 구성

### 2.4 Android 홈 위젯
- 5x1 위젯 제공
- 위젯 표시 정보
  - 날씨
  - 기온
  - 체감온도
  - 야외 점수
  - 운세 아이콘
  - 날짜, 요일, 시간
- 위젯 날씨 갱신 지원
- 위젯 시간은 휴대폰 현재 시간을 기반으로 표시
- 위젯 날씨 아이콘 PNG 자산 적용
- 위젯 설치 안내
  - 앱 첫 실행부터 위젯 설치 안내 여부 확인
  - 버전/빌드별 안내 처리 키 사용
  - 업데이트 후에도 새 버전 기준으로 안내 여부 재판단
  - Android 시스템의 위젯 ID가 남아 있어도 안내창을 차단하지 않도록 개선

### 2.5 앱 업데이트
- 앱 실행 시 최신 버전 확인
- 업데이트 필요 시 안내창 표시
- Play Store 또는 인앱 업데이트 경로로 이동

### 2.6 광고
- 하단 배너 광고 영역 유지
- 앱 종료 확인 후 조건 충족 시 전면 광고 표시 로직 포함

### 2.7 설정 및 앱 정보
- 앱 정보 페이지 제공
- 홈페이지, 개인정보 처리방침, 문의 이메일 등 항목 연결
- 앱 버전 표시
- 설정 화면에서 디버그성 항목 제거

## 3. 중요 수정사항 요약

### 3.1 날씨 데이터 안정화
- Open-Meteo 중심 흐름을 제거하고 한국 기준 기상청 API 중심으로 정리
- 기상청 초단기실황, 초단기예보, 단기예보, 중기예보 조합 사용
- 시간별 예보 24시간 표시 보강
- 주간 예보 7일 표시 보강
- 예보 파싱 실패 시 기존 정상 캐시 유지
- 앱 진입 시 캐시를 먼저 보여주고 이후 비차단 갱신하도록 변경

### 3.2 야간 날씨 아이콘 수정
- 기존 문제: 시간별 예보는 야간 아이콘이 적용되지만 현재날씨/위젯은 야간 아이콘이 제대로 적용되지 않음
- 원인: 현재날씨와 위젯 저장 경로에서 야간 판정값 전달이 일관되지 않음
- 수정: `20:00~05:59`를 야간, `06:00~19:59`를 낮으로 통일
- 현재날씨, 시간별 예보, 위젯 저장 키 모두 동일 기준을 사용하도록 정리

### 3.3 강수량 표시 개선
- 현재날씨 박스에 강수량 표시 추가
- 강수량이 0mm인 경우 현재날씨 박스에서 숨김
- 주간 예보에 예상강수량 표시 추가
- 단위는 기상청 강수량 기준에 맞춰 mm 사용

### 3.4 위젯 개선
- 위젯 레이아웃을 5칸 구조로 재정리
- 날씨/야외/운세/날짜시간 영역의 크기와 정렬 반복 조정
- 위젯 날씨 아이콘 깨짐 문제 해결을 위해 위젯 전용 PNG 자산 적용
- 위젯 갱신 시 앱 야외 점수 캐시를 사용하도록 수정
- 위젯 설치 안내 로직 개선
  - 단일 handled 키에서 버전/빌드별 handled 키로 변경
  - 첫 실행부터 안내 여부 확인
  - 시스템 위젯 ID 잔존으로 인한 안내 차단 문제 완화

### 3.5 뒤로가기 및 종료 처리
- 홈 화면에서 뒤로가기 시 앱 종료 확인창 표시
- 다른 페이지에서 뒤로가기 시 홈으로 이동
- 종료 확인 후 조건 충족 시 전면 광고 표시
- 광고 표시 후 앱 종료 흐름 보완

### 3.6 UI/UX 개선
- 전체 디자인을 날씨 중심의 프리미엄 톤으로 재정리
- 날씨를 메인, 운세를 보조 정보로 배치
- 카드 기반 화면 구성 정리
- 하단 카테고리바 강조 상태 개선
- 날씨 페이지의 작은 정보 박스를 더 압축하고 수치를 제목 우측으로 이동
- 현재날씨 박스의 위치/날씨/강수량 정보 가독성 개선

### 3.7 데이터 출처 및 정책
- 기상청 공공누리 출처 표시 반영
- 에어코리아 대기질 출처 표시 반영
- 앱 정보 페이지의 개인정보 처리방침 등 링크 항목 정리

## 4. 프로덕션 출시 전 확인 필요 항목
- Google Play Console에 등록된 최신 버전 코드가 40보다 낮은지 확인
- 스토어 등록용 AAB 파일 확인

- 실제 기기에서 확인할 항목
  - 앱 첫 실행 위젯 설치 안내창
  - 날씨 캐시 우선 표시
  - 현재날씨 야간 아이콘
  - 위젯 야간 아이콘
  - 시간별 예보 24시간
  - 주간 예보 7일
  - 강수량 0mm 숨김
  - 야외 점수와 위젯 야외 점수 일치
  - 업데이트 확인창
  - 뒤로가기 종료 확인창
